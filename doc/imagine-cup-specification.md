## 1. プロジェクト概要

**プロジェクト名**：ferro
**カテゴリ**：AI / Mixed Reality / Creative Technology
**プラットフォーム**：Web（Next.js + Three.js）、将来的にネイティブ拡張も想定

### 一文サマリー

ferroは、音に反応して静かに動く存在です。
音楽とともに在り、使う人が自分の感情に気づくきっかけをつくります。

## 2. 問題定義（Imagine Cup）

多くの人は、仕事・勉強・日常生活の中で常にプレッシャーを感じています。
しかし、自分の感情に気づいたり、整理したりする時間はほとんどありません。

現在のデジタルツールの多くは、

- 入力を求める
- 評価・分析を行う
- 明確な操作や成果を要求する

という前提で作られています。

### 課題

ユーザーを操作させたり、評価したりせず、
ただ「そばに在る」ことで感情への気づきを支える体験は作れないだろうか？

## 3. 解決策

ferroは、磁性流体（ferrofluid）から着想を得た抽象的なデジタルオブジェクトです。

- 音楽や環境音にリアルタイムで反応する
- 音の特徴を、有機的な動きとして可視化する
- 言葉を話さず、指示も評価もしない

意味や感情は、ユーザー自身の解釈によって生まれます。

## 4. コンセプトと思想

### 基本原則

- 近くにあるが、干渉しない
- 導かず、反応する
- 感情は説明されるものではなく、感じられるもの

### 影響を受けた思想

- **九鬼周造『いきの構造』**
  → 言語化できないが、確かに存在する感覚の構造

- **イヴァン・イリイチ「コンヴィヴィアルな道具」**
  → 使う人の自由を奪わない道具

### デザインスタンス

ferro自身が感情を理解するわけではない。
感情が生まれるのは、それを見ている人間の側である。

## 5. 想定ユーザーと利用シーン

### 想定ユーザー

- 音楽を聴きながら作業する人
- クリエイター、デザイナー、開発者
- 言葉を使わない、静かな癒しを求める人

### 利用シーン

- **仕事・集中時間**
  作業中、背景音楽とともにferroが静かに動く

- **共有の時間**
  家族や友人と音楽を聴く空間で、一緒に反応する存在として在る

## 6. 機能（MVP）

### 音声入力

- マイク入力（ユーザー許可制）
- ブラウザタブ音声（対応環境のみ）
- サンプル音源の再生

### 音声解析

- 音量・エネルギー
- 周波数帯（低・中・高）
- 時間変化（揺らぎ）

### 視覚表現

- ノイズを用いた有機的変形
- スケールや動きの変化
- 色・質感の変化

## 7. AI・技術スタック

### フロントエンド

- Next.js
- Three.js / WebGL
- Web Audio API

### AI / クラウド（Imagine Cup 想定）

**メインAI（デフォルト）：Azure AI**

- **Azure AI Speech**
  → 音声特徴量の解析

- **Azure OpenAI**
  → 音の状態を抽象化し、視覚パラメータに変換

**サブAI（切り替え可能）**

- OpenAI API（比較・検証用）
- その他将来的なAI（Gemini 等）

## 8. AI切り替え設計（管理画面）

### 基本方針

- Azure AIをデフォルトとする
- AIは体験の前面に出さない
- ユーザーはAIの存在を意識しなくてよい

### 管理画面（開発者向け）

使用するAIを切り替え可能：

- Azure AI（Default）
- OpenAI API
- None（AIを使わないローカル解析）

### 切り替え対象

- 音声解析ロジック
- 抽象化レイヤーの有無
- パラメータ補正ロジック

### 目的

- Imagine Cup要件（Microsoft技術活用）への対応
- AI挙動の比較・検証
- 将来的な拡張性の確保

**AIは入れ替え可能だが、ferroの在り方は変わらない。**

## 9. なぜ Imagine Cup なのか

### AIと人間中心設計の融合

- 医療化しないウェルビーイング
- 話さない、目立たないAIの活用

ferroは、AIを次のように使う可能性を示します。

- 静かで
- 尊重的で
- 感情に寄り添う存在としてのAI

## 10. 倫理的配慮

- 感情のラベリングを行わない
- ユーザープロファイリングをしない
- データを保存しない（デフォルト）

ユーザーは常に主導権を持ちます。

## 16. 開発上の注意（最重要）

- ferroに人格を持たせない
- 正解や評価を与えない
- 感情を決めつけない

**ferroは、反応するだけの存在である。**
**感情が生まれるのは、人間の側である。**
